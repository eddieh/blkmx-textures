▓▒░ BLKMX Textures ░▒▓
======================

Block textures remixed. A texture pack for Minetest & MineClone2.

All the best assets from texture packs, cherry picked, altered, and
remixed.


Sizes
-----

block texture 16x16
item  texture 16x16

entity skins are multiples of 16n x 16m: 32x16, 64x32, 32x32 etc.
